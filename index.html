BRIX RL

A modular, transparent, and scalable Reinforcement Learning framework designed with simplicity in mind.

View on GitHub

⚡ Envs

Environment settings, vectorized configurations, and gym wrappers to handle state and reward pipelines efficiently.

🧠 Models

Core neural network architectures including shared representation MLPs and decoupled Policy/Value networks.

💾 Buffers

Experience replay, rollout buffers, and trajectory management for off-policy and on-policy data collection.

⚙️ Algorithms

Standardized algorithms like PPO inheriting from unified base classes for clean training and inference loops.

Quickstart

Here's an abstracted example of how the framework components hook together:

import gymnasium as gym
from brix.envs.wrappers import make_env
from brix.models.mlp import PolicyValueMLP
from brix.buffers.replay_buffer import RolloutBuffer
from brix.algorithms.base import BaseAgent

# 1. Setup Environment
env = make_env("CartPole-v1")

# 2. Setup Neural Network Architecture
model = PolicyValueMLP(
    input_dim=env.observation_space.shape[0],
    action_dim=env.action_space.n
)

# 3. Setup Buffer for Experience Collection
buffer = RolloutBuffer(capacity=2048, state_dim=env.observation_space.shape)

# 4. Initialize Algorithm
agent = BaseAgent(env=env, model=model, buffer=buffer)

# 5. Train
for epoch in range(100):
    agent.train_step()
